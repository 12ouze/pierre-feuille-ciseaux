# 🎮 Pierre-Feuille-Ciseaux

Un projet pédagogique d'apprentissage du code en 3 langages : Python (CLI) et HTML/CSS/JS (web).

## 🎯 Ce que contient ce projet

- **`pfc.py`** — le jeu en version ligne de commande, en Python pur.
- **`index.html`** — la structure de la version web.
- **`style.css`** — la mise en forme et le design de la page.
- **`script.js`** — la logique du jeu côté navigateur.
- **`CLAUDE.md`** — les consignes destinées à Claude Code pour travailler sur le projet.
- **`LEXIQUE.md`** — un glossaire de termes techniques expliqués simplement.

## 🐍 Lancer la version Python

```bash
python3 pfc.py
```

## 🌐 Lancer la version web

Il suffit de double-cliquer sur le fichier `index.html` pour l'ouvrir dans votre navigateur.

## 📚 Apprentissage

Ce projet a été créé en duo avec **Claude Code**, dans un objectif d'apprentissage par la pratique. Le fichier `LEXIQUE.md` rassemble **110 termes techniques** expliqués en français simple, pensés pour les débutants.

## ⚙️ Tech stack

- Python 3
- HTML
- CSS
- JavaScript (vanilla, sans framework)
